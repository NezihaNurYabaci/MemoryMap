#  MemoryMapp - Keep Your Memories Alive

MemoryMapp is a location-based digital diary application developed with **Kotlin** and **Jetpack Compose**. This project was created as part of my Erasmus+ studies at **West Pomeranian University of Technology** and refined during my stay in Poland.

---

##  Key Features

* **Interactive Map:** Explore and pin memories on an interactive map using OSMDroid.
* **Anniversary Notifications:** The app automatically checks your history and notifies you if you were at a specific location exactly **one year ago today**.
* **Firebase Integration:** Real-time data synchronization with Firebase Realtime Database and secure authentication via Email or Google.
* **Location Intelligence:** Automatically fetches addresses from coordinates using Geocoding.
* **Modern UI:** A clean, user-friendly interface designed with Material3, featuring consistent blue branding.

---

##  Tech Stack

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Database:** Firebase Realtime Database
* **Maps:** OSMDroid
* **Architecture:** MVVM (ViewModel, StateFlow)
